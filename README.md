CampusBuddy (Backend)-
CampusBuddy is a smart campus assistance platform that helps students access academic tools, notes, and AI-based features in one place.
This backend is built using Node.js, Express, and MongoDB, with OpenAI integration for intelligent responses and chat support.

Features-
RESTful API built with Express.js
MongoDB Atlas for database connectivity
Integration with OpenAI API
Secure environment variables with .env
Clean and modular backend structure

Tech Stack
Below is an overview of the technologies used in the backend:-
Node.js — Provides the JavaScript runtime environment for building scalable server-side applications.
Express.js — Handles routing and server logic through a fast and minimalist web framework.
MongoDB Atlas — Serves as the cloud-based NoSQL database for storing user and app data.
Mongoose — Offers a simple, schema-based solution to model application data in MongoDB.
OpenAI API — Powers AI-driven chat and response functionalities within the platform.
dotenv — Manages sensitive configuration data using environment variables securely.
